# README

Simple Blog using Ruby on Rails Full Stack and MySQL DataBase, with **Ruby** in version 2.6.

## Dependencies

*Ruby 2.6*
*Ruby on Rails 5.5.2*
*MySQL 5.7.24*

## Build

**1. Install gems**

```
bundle install
```

**2. Create database**

```
rake db:create
```

**3. Run migrations**

```
rake db:migrate
```

**4. Start server**

```
rails s -p 3000 -b '0.0.0.0'

```
